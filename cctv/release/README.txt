============================================
  MomayBUU CCTV Relay — ไฟล์ติดตั้ง
============================================

ในโฟลเดอร์นี้มี 2 แบบ เลือกใช้อันใดอันหนึ่ง:

1) MomayRelay-Setup.exe   (แนะนำ)
   - ตัวติดตั้งจริง ดับเบิลคลิกแล้วทำตามขั้นตอน
   - ติ๊ก "เปิดอัตโนมัติเมื่อเข้า Windows" ได้ → relay รันเองตอนเปิดเครื่อง
   - เครื่องปลายทางไม่ต้องลง Python

2) MomayRelay.zip          (แบบพกพา)
   - แตกไฟล์แล้วดับเบิลคลิก MomayRelay.exe ได้เลย ไม่ต้องติดตั้ง

--------------------------------------------
หลังเปิดโปรแกรม
--------------------------------------------
- เบราว์เซอร์จะเปิดหน้า Control Panel: http://127.0.0.1:8090
- ใส่รหัสเข้า Control Panel: 240124
- ตรวจ RELAY_KEY ให้ตรงกับ cctv server แล้วกด Start
- เพิ่ม/ตั้งค่ากล้องทำที่หน้า /settings ของ Momay (ไม่ใช่ที่นี่)
  • เปิด "ตรวจจับคน" รายกล้อง → จะนับคน + % พื้นที่ ส่งเข้า heatmap/เกจ

* เครื่องที่รันต้องอยู่วงเน็ตเดียวกับกล้อง (เข้าถึง RTSP ได้)
* รัน relay 1 ตัวต่อ 1 ไซต์ (ดึงทุกกล้องจาก registry เอง)
